# Manual Verification Checklist

1. Run `npm run dev` in `stc-web/`.
2. Open `http://localhost:3000/` and confirm the pixel-close homepage renders.
3. Open `http://localhost:3000/v2` and confirm the componentized homepage renders.
4. Verify all sections load and the logo is the updated `stc-logo-horizontal.png` on both routes.
5. Compare `/` against the original mockup for visual fidelity.
