# STC Imagery and Iconography Standards

Purpose: keep visual assets authentic, cohesive, and trust-building.

## 1) Imagery Principles
- Show real people and real treatment environments.
- Prioritize emotional safety, dignity, and calm.
- Avoid stereotypical “corporate healthcare” stock compositions.

## 2) Image Style by Section

### Hero
- One clear focal subject or environment.
- Natural light, low visual noise.
- Maintain copy readability (overlay/gradient if needed).

### Services
- Contextual environments (group room, therapy space, clinician interaction).
- Avoid staged handshakes/boardroom imagery.

### Testimonials
- Candid, neutral lighting, supportive facial expression.
- Consistent crop ratio and background softness.

### Locations
- Exterior/interior orientation imagery with clear context.
- Include practical trust signals (clean, safe, welcoming environment).

## 3) Cropping + Treatment
- Prefer 4:3 or 3:2 for cards; 16:9 for hero media.
- Avoid excessive filters; use subtle color correction only.
- Keep skin tones natural.

## 4) Iconography System
- Style: clean line icons with rounded corners.
- Stroke width: 1.75-2.0px equivalent.
- Corner treatment: soft/rounded, not sharp geometric spikes.
- Visual tone: clear, approachable, clinical.

## 5) Color Use for Icons
- Default icon color: `#1F1CDE`
- Accent icon moments: `#22B610`
- Avoid mixed rainbow icon palettes.

## 6) Do / Don’t
Do:
- keep icon set stylistically consistent
- use icons to clarify, not decorate
- use authentic photography

Don’t:
- use generic “people pointing at laptop” stock
- mix outlined and heavy-filled icon families randomly
- place text on low-contrast photos without overlays
