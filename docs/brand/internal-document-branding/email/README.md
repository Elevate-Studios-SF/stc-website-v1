# STC Internal Email Templates

## Files
- `session-reminder-email.html`: Branded reminder email template based on STC brand guide colors and typography.

## Brand Tokens Used
- Deep Indigo: `#1F1CDE`
- Soft Blue: `#D4E3FC`
- Healing Green: `#22B610`
- Treatment Center Logo Color: `#5F7FB0`
- Font stack: `Tahoma, Verdana, Segoe UI, Arial, sans-serif`

## Notes
- Template uses table-based email markup for broad email client compatibility.
- Update recipient name, session details, and contact details before sending.
