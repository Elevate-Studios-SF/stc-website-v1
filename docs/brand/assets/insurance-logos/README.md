# Insurance Logos

Use this folder for approved insurance brand assets used in the "Insurance Accepted" block.

## Target carriers
- Blue Cross Blue Shield
- Presbyterian
- New Mexico Health Connections
- Medicaid Centennial plans

## Source policy
- Prefer official media/brand kits from each insurer.
- Do not hotlink third-party image URLs in production.
- Store local approved files and optimize them (SVG preferred, PNG fallback).

## Recommended file names
- insurance-bcbs.svg
- insurance-presbyterian.svg
- insurance-nm-health-connections.svg
- insurance-medicaid-centennial.svg

## Fallback
If official logos are not available yet, use text chips in UI and mark as `asset pending`.
