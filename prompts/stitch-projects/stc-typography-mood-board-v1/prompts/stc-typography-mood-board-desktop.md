Create a desktop-first **Typography Mood Board** for STC (Solutions Treatment Center) that explores multiple clean, modern heading-font pairings with `Tahoma` body text, then recommends the best options for production use.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first (with mobile previews)
- Brand style: Clinical, calm, trustworthy, modern
- Theme direction: Light clinical
- Background: Soft Blue (#D4E3FC)
- Surface: White (#FFFFFF)
- Primary Accent/CTA: Healing Green (#22B610)
- Secondary Accent/Structure: Deep Indigo (#1F1CDE)
- Color usage ratio: 60% Soft Blue, 30% White, 10% Green/Indigo accents
- Approved pairings: #D4E3FC background + #101828 text; #FFFFFF surface + #101828 text; #22B610 CTA + #FFFFFF text; #1F1CDE for headings/dividers/nav accents
- Text on dark: White (#FFFFFF)
- Text on light: Near Black (#101828), Secondary (#344054)
- Body/UI font (required): Tahoma
- Body stack: Tahoma, Verdana, Segoe UI, Arial, sans-serif
- Keep body text at 16px minimum
- Use one heading family per variation (no mixed heading families inside a variation)
- Maintain high contrast and healthcare-grade readability
- Buttons: 12px radius, clear focus/hover states

**TYPOGRAPHY SYSTEM (REQUIRED):**
- Primary UI/Body font: Tahoma
- Include these heading pairings:
  1. Georgia + Tahoma (trustworthy, familiar)
  2. Fraunces + Tahoma (modern, authoritative)
  3. Merriweather + Tahoma (calm, long-form)
  4. Source Serif 4 + Tahoma (structured, professional)
  5. IBM Plex Serif + Tahoma (clean, modern recommendation candidate)
  6. Spectral + Tahoma (modern editorial recommendation candidate)

**Page Structure:**
1. **Header:** “STC Typography Mood Board” title, short purpose statement, and legend.
2. **Variation Grid (6 cards):** One card per pairing with:
   - Font names and intended tone
   - H1/H2/H3/Body/Caption specimen text
   - Sample UI snippet (nav label, button label, form label, paragraph)
   - Quick readability tags: “Clarity”, “Trust”, “Modernity” (1–5 scale visuals)
3. **Type Scale Preview:**
   - Desktop: H1 56/62, H2 40/48, H3 30/38, H4 24/32, Body Large 18/30, Body 16/26, Caption 14/22
   - Mobile: H1 40/46, H2 32/38, H3 24/30, H4 20/26, Body 16/24, Caption 14/20
4. **Recommendation Panel:**
   - Rank top 3 pairings for STC website use
   - Provide short rationale for each (readability, professionalism, modern feel, healthcare trust)
   - Mark one as “Primary Recommendation” and one as “Campaign/Landing Recommendation”
5. **Usage Rules Panel:**
   - Tahoma for all navigational/functional UI text
   - 16px minimum body size
   - Use semibold/700 sparingly
   - One heading family per page

**Visual Direction:**
- Clean, high-end clinical aesthetic with generous whitespace
- Strong hierarchy, clear typography-first composition
- Professional mood board layout (not a marketing landing page)
- Distinct card styling so differences between pairings are immediately visible
