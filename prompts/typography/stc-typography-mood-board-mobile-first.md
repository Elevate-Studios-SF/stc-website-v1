Create a **mobile-first Typography Mood Board** for STC (Solutions Treatment Center) that compares clean, modern heading-font options paired with `Tahoma` body text and recommends the strongest pairings for production.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Mobile-first (with desktop companion preview)
- Brand style: Clinical, calm, trustworthy, modern
- Theme direction: Light clinical
- Background: Soft Blue (#D4E3FC)
- Surface: White (#FFFFFF)
- Primary Accent/CTA: Healing Green (#22B610)
- Secondary Accent/Structure: Deep Indigo (#1F1CDE)
- Color usage ratio: 60% Soft Blue, 30% White, 10% Green/Indigo accents
- Approved pairings: #D4E3FC background + #101828 text; #FFFFFF surface + #101828 text; #22B610 CTA + #FFFFFF text; #1F1CDE for headings/dividers/nav accents
- Text on dark: White (#FFFFFF)
- Text on light: Near Black (#101828), Secondary (#344054)
- Body/UI font (required): Tahoma
- Body stack: Tahoma, Verdana, Segoe UI, Arial, sans-serif
- Keep body text at 16px minimum
- Use one heading family per variation (no mixed heading families inside a variation)
- Maintain high contrast and healthcare-grade readability

**TYPOGRAPHY SYSTEM (REQUIRED):**
- Primary UI/Body font: Tahoma
- Include these heading pairings:
  1. Georgia + Tahoma
  2. Fraunces + Tahoma
  3. Merriweather + Tahoma
  4. Source Serif 4 + Tahoma
  5. IBM Plex Serif + Tahoma
  6. Spectral + Tahoma

**Page Structure:**
1. **Top Header:** “STC Mobile Typography Mood Board” with short purpose statement.
2. **Scrollable Variation Stack (6 cards):** One card per pairing with:
   - Font names + tone
   - H1/H2/H3/Body/Caption specimens
   - Mini mobile UI sample (nav label, button, form label, paragraph)
   - Readability indicators: Clarity / Trust / Modernity (1–5)
3. **Type Scale Panel (Mobile Priority):**
   - Mobile: H1 40/46, H2 32/38, H3 24/30, H4 20/26, Body 16/24, Caption 14/20
   - Desktop reference: H1 56/62, H2 40/48, H3 30/38, H4 24/32, Body Large 18/30, Body 16/26, Caption 14/22
4. **Recommendation Panel:**
   - Rank top 3 pairings for STC
   - Label one “Primary Site Pairing” and one “Campaign Pairing”
   - Include concise rationale focused on mobile readability + modern clinical trust
5. **Usage Rules Panel:**
   - Tahoma for all functional UI text
   - 16px minimum body size
   - One heading family per page
   - Use semibold/700 sparingly

**Visual Direction:**
- Premium clinical style with generous spacing and clear hierarchy
- Typography-first composition optimized for phone screens
- Distinct cards so pairing differences are easy to compare at a glance
